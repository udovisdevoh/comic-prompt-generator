Choisir le mode de génération :

Paramètres de la Bande Dessinée (Communs à tous les modes)

Mode par Étapes

Étape 1 : Définition de l'Album/Histoire/Gag

Étape 2 : Structure des Pages/Strips

Étape 3 : Génération Détaillée des Pages/Strips

Étape 4 : Compilation Finale (Texte)

Mode Prompt Unique (Sortie Texte)

Ce mode génère un seul prompt demandant à l'IA de produire l'intégralité du contenu textuel (définition, résumés, détails de chaque case pour toutes les pages/strips). Attention, cela peut générer un prompt et une réponse très longs.

Mode Storyboard (Sortie HTML/SVG)

Ce mode génère un prompt unique demandant à l'IA de produire un storyboard complet au format HTML avec des dessins SVG schématiques. La qualité des SVG dépendra fortement des capacités de l'IA. Le résultat peut être très long.
